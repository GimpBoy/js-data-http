Quick Start

bower install --save js-data js-data-http or npm install --save js-data js-data-http.

Load js-data-http.js after js-data.js.

var adapter = new DSHttpAdapter();

var store = new JSData.DS();
store.registerAdapter('http', adapter, { default: true });

// "store" will now use the http adapter for all async operations

Contributing

First, feel free to contact me with questions. Mailing List. Issues.

  1. Contribute to the issue that is the reason you'll be developing in the first place
  2. Fork js-data-http
  3. git clone https://github.com/<you>/js-data-http.git
  4. cd js-data-http; npm install; bower install;
  5. grunt go (builds and starts a watch)
  6. (in another terminal) grunt karma:dev (runs the tests)
  7. Write your code, including relevant documentation and tests
  8. Submit a PR and we'll review
